## ⚡ Overview

**visupals.** adalah platform portofolio digital yang dirancang untuk merepresentasikan identitas visual **Naufal Dzaky S.** sebagai seorang *Content Creator* dan mahasiswa *Computer Engineering* di **Universitas Brawijaya**.

## 🎨 Visual Identity
Situs ini dibangun dengan filosofi desain *Kinetic Minimalist*, menggunakan:
- **Primary Color:** Orange Brand (`#f97316`).
- **Typography:** Plus Jakarta Sans.
- **UI Style:** Bento-style cards & Mesh Gradients.

## 🛠️ Tech Stack
- **Framework:** Laravel 12.x
- **Admin Panel:** Filament v3 (Custom Makeover).
- **Styling:** Tailwind CSS.
- **Interactivity:** Alpine.js.
- **Database:** MySQL.
